# Agentic AI: The Architectural Foundations of Modern AI Agents

This repository contains the slides and materials for the seminar **"Agentic AI: The architectural foundations of modern AI agents"** held at the **University of Urbino** for the course *Foundations of Artificial Intelligence (Fondamenti di Intelligenza Artificiale)*.

## Overview

The seminar explores the evolution of Large Language Models (LLMs) from simple text generators to autonomous agents capable of reasoning, planning, and interacting with the external world.

### Key Topics
- **What is an LLM agent?** — Defining autonomy, reasoning, and tool use.
- **Core Components** — The "Brain" (LLM), Memory systems (Short vs. Long term), and Planning strategies.
- **Tools & Interoperability** — Function calling and the Model Context Protocol (MCP).
- **Modern Frameworks** — Building stateful agents with LangChain and LangGraph.
